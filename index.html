    ❓ How to use Exchel — every tool

    File

    🗋 NewStart a blank workbook (asks first — unsaved work is lost).
    📂 OpenOpen an .xlsx, .xls or .csv file from your computer. Formulas inside are imported and recalculated.
    💾 Save .xlsxDownload the whole workbook as a real Excel file — formulas are preserved, so it keeps calculating in Excel.
    ⬇ CSVDownload only the current sheet as plain CSV (values, no formulas).

    Edit

    ↶ Undo / ↷ RedoStep backwards / forwards. Shortcut: Ctrl+Z / Ctrl+Y.
    ✕ ClearEmpty the selected cells. Shortcut: Delete.
    Right-clickExcel-style menu on any cell or header: cut / copy / paste, insert or delete rows and columns, clear contents. Formulas adjust automatically when rows/columns move.
    − Row / − ColDelete the selected row(s) or column(s) from the toolbar.
    SelectingClick a cell and drag to select a range. Click a row number or column letter to select the whole row/column — drag or Shift+click across headers for several.
    Column widthDrag the edge of a column letter to resize, double-click it to auto-fit that column, or use ↔ Fit Cols for all. Clean Data and opening a file auto-fit automatically.
    ▾ Filter dropdownsEvery column letter has a ▾ — click it to tick/untick the values you want visible, exactly like Excel's AutoFilter. Filtered columns show a green ▾. "Clear filter" or ⊘ Show All removes it.
    𝐁 Bold / ▦ BordersFormat the selected cells (bold text or dark borders). Shortcut for bold: Ctrl+B. On-screen formatting only — values and formulas are what get saved to .xlsx.

    Automate

    ⚡ Auto ReportOne click does everything: cleans the data, writes an Analysis sheet with statistics for every column, auto-detects the label column and the number column, builds a SUM pivot, and opens a chart of it — ready to download as PNG.
    Σ AutoSumSelect numbers, click — a =SUM(…) formula is written in the cell below the selection.
    ⤓ Fill DownCopies the top row of the selection into every row below it. Formulas shift automatically (A1 → A2 → A3…; use $A$1 to lock). Shortcut: Ctrl+D.
    ⤏ Fill RightSame, but copies the left column across. Shortcut: Ctrl+R.
    A→Z / Z→A SortSelect whole rows of data, click — rows are sorted by the first selected column.
    🔎 ReplaceFind & replace text across the whole current sheet.
    ⊞ Pivot TableSelect your data (or just click one cell — the whole sheet is used). First row must be headers. Pick Rows / Columns / Values / Calculation, and a finished pivot with totals appears on a new sheet.
    🧹 Clean DataOne click: trims spaces, turns text like "1,250" into real numbers, deletes blank rows and duplicate rows. Fully undoable with Ctrl+Z.

    Analyze

    📊 ChartSelect data (labels in the first column, numbers next to them) and click. Switch between bar / line / pie / doughnut and download as PNG.
    🎨 HighlightConditional formatting: colour cells that are greater / less than a value, equal to something, contain text, are duplicates, top-10, or blank.
    ⏳ FilterClick a cell in the column to filter, then enter a condition like >100, =South or plain text (contains). Non-matching rows are hidden on screen.
    ⊘ Show AllRemoves the filter — all rows visible again.
    ✂ Split ColText to Columns: splits the active column by a separator (comma, space, dash…) into the columns beside it.
    📚 Merge FilesPick several .xlsx / .csv files at once — each becomes a sheet tab, and optionally all rows are stacked into one "Combined" sheet (headers kept once). Perfect for combining monthly reports.
    📈 AnalyzeOne click writes a summary report sheet: filled cells, numbers, blanks, distinct values, sum, average, min and max for every column.
    + Rows… / + Cols…Adds rows or columns to the grid — a box asks how many you need (2, 50, anything).

    Formulas

    Formula bar (fx)Click a cell, type =SUM(A1:A10) and press Enter. While typing a formula you can click any cell to insert its reference, just like Excel.
    Name boxType a cell address (e.g. C10) and press Enter to jump there.
    AutocompleteType = and a few letters in a cell — matching functions pop up like in Excel. Press Tab or click to complete.
    Sidebar listClick any function to see its syntax, description and a worked example — then press "Insert into cell". Double-click inserts straight away.

    Keyboard shortcuts

    Arrows / Enter / TabMove around (Shift+Arrows extends the selection).
    F2Edit the active cell.
    Ctrl+C / Ctrl+VCopy / paste — works with real Excel and Google Sheets via the clipboard.
    Ctrl+ASelect everything.
